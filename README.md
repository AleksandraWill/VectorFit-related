This repository contains adapted versions of the VectorFit scripts fit_beads_aberrations.m, get_NAT_inversion.m, several set_parameters.m variants, and segmentation_beads_difference_uniform_cropped.m, along with results used for aberration characterization of different image-splitter configurations. The original VectorFit framework was developed by the ImPhys group at TU Delft and is distributed under the Apache-2.0 license: https://gitlab.tudelft.nl/imphys/ci/vectorfit
